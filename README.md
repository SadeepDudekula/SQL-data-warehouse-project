# Data Warehoouse and Analytics Project

## Overview
This project shows how to build a **SQL Data Warehouse** for analytics.  
It focuses on schema design, data transformation, and querying structured data.

## What This Project Includes
- Fact and dimension table design  
- SQL-based ETL process  
- Analytical queries for reporting  

## Objective
Develop SQL-based analytics to deliver detailed insights into :
**Customer Behaviour**
**Product Performance**
**Sales Trends**

These insights empower stakeholders with key bussiness matrics, enabling strategic decision making.
